# ライブラリを導入して使用してみる

headlessUI + TailwindCSS を React プロジェクトにダウンロードし、アコーディオン UI を作成。

1. 質問をクリックすると質問の答えがアコーディオンで出てくる
2. 質問をホバーすると色が変わる

## project の使い方

```

git clone https://github.com/aya030/react-headless.git
cd react-headless
npm install

```
